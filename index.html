PORQUE APRENDER CSS...

Al igual que el HTML, el CSS es el lenguaje más importante con relación al diseño web. Toda persona que esté deseando crearse una página web con estilo y con gran diseño no puede no conocer este lenguaje. Al mismo tiempo, contar con conocimientos básicos de CSS es muy útil a la hora de trabajar con editores gráficos. A la hora de trabajar con páginas web, hay una diferencia muy importante entre solamente saber cómo manejar herramientas como WordPress o Joomla y tener los conocimientos básicos de este tipo de tecnología. Es cierto que estas herramientas nos facilitan terriblemente el trabajo, porque nos abstraen de las tecnologías subyacentes a la página web. Pero conocer las bases, el HTML y el CSS, son fundamentales para un diseñador. Podemos aplicar el CSS en muchas herramientas originales. Una de ellas es el diseño de un email para clientes. Las campañas que se realizan a través de los emails es una muy buena forma de poder hacerte publicidad. Un email con un gran diseño relacionado a tu empresa generará una fuerte impresión en tus clientes. Otro uso es dentro de tu página web. Si bien muchos sitios están construidos en Wordpress, y probablemente el de tu empresa lo esté también, con algo de conocimiento en HTML y CSS, puedes realizar pequeños ajustes que generarán grandes cambios a nivel visual. ¡Y todo esto con un mínimo de conocimiento, sin ser un experto!

OBJETIVOS GENERALES

Este curso está dirigido a desarrolladores y diseñadores web tanto nuevos como experimentados que buscan ampliar sus habilidades para incluir los próximos estándares web. A través de laboratorios prácticos y ejercicios, el alumno aprenderá los fundamentos de la creación de páginas web modernas con las normas/estándares actuales. El alumno adquirirá, entre otras, destrezas que le permitirán:

• Entender cómo construir aplicaciones web basadas en estándares como HTML (lenguaje de marcas de hipertexto), CSS (hojas de estilo en cascada)
• Entender la arquitectura cliente-servidor y las características de los servidores web, considerando las peculiaridades de los clientes asociados.

OBJETIVOS ESPECIFICOS

1. Aprender y crear la estructura de HTML5

2. Maquetar un archivo HTML semánticamente

3. Agregar imágenes en su diferentes formatos

4. Usar selectores de CSS

5. Estilos de CSS
